### 🤱 MaternaRisk AI — Maternal Health Risk Classifier
> *Helping Lady Health Workers in rural Pakistan screen pregnant women for risk*

| Detail | Info |
|--------|------|
| **Model** | XGBoost with 5-fold cross-validation (~0.86 CV score) |
| **Performance** | ~87% weighted F1-score |
| **Dataset** | Merged UCI + Mendeley maternal health datasets |
| **Balancing** | SMOTETomek (handles class imbalance) |
| **Features** | Blood sugar, Systolic/Diastolic BP, Heart rate, Age |
| **Deployment** | 4-page Streamlit app hosted on Google Colab |
